# Grant Application Checklist 

- [ ] Open-source license present (`LICENSE`)
- [ ] Contract source and tests available (`contracts/wave-contract`)
- [ ] SDK buildable (`sdk/`)
- [ ] CI configured and passing (`.github/workflows/ci.yml`)
- [ ] Security policy published (`SECURITY.md`)
- [ ] Governance model documented (`GOVERNANCE.md`)
- [ ] Funding transparency documented (`FUNDING_TRANSPARENCY.md`)
- [ ] Drips config template included (`drips-stream-config.json`)
- [ ] Roadmap with measurable milestones (`docs/ROADMAP.md`)
- [ ] Maintainer process and succession documented (`docs/MAINTAINER_GUIDE.md`)
