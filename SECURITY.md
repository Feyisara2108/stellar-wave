# Security Policy

## Supported Versions

This project currently supports the `main` branch and latest tagged release.

## Reporting a Vulnerability

Please report vulnerabilities privately to: `security@waveproject.dev`.

Include:
- affected component
- reproduction details
- impact assessment

Initial maintainer response target: 24 hours.

## Security Boundaries

- Contract mutating operations must enforce explicit authorization.
- Emergency pause mechanism exists for incident containment.
- No private keys or secrets may be committed to this repository.
